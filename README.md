### Contribution

```bash
cd path/to/yhttp
pip install -e .
pip install -r requirements-dev.txt
```

#### Running tests

```bash
pytest
```

#### Coverage

```bash
pytest --cov=yhttp
```

#### Documentation

```bash
cd sphinx
make doctest
make html
make livehtml
```
